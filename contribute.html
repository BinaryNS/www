Contribute to BinaryNS.github.io/www

How do contribute models?

The short answer is: you can't right now. We are in the process of setting up this site, so there isn't any infrastructure for that yet. Having said that, if you are really interested in contributing now, please don't hesitate to contact us and we go from there.

How do I make changes to this website?

This website it hosted by Github, and so is the repository backing it: https://github.com/BinaryNS/www. One way to contribute changes here would be to fork and to create a pull request. The other option is to request write access to this repository.

How do I make changes to the scientific software used here?

That depends on the particular piece of software. Most of the software used here is developed and maintained elsewhere, which means you have to go and ask there. However, there are some scripts used to run LORENE to get input files for the Einstein Toolkit, and those are maintained here, and you can find them on github: https://github.com/BinaryNS/LoreneBNScripts. One way to contribute changes here would be to fork and to create a pull request. The other option is to request write access to this repository.
